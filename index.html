APP 下載與帳號登入

📲 下載安裝

  • iOS：App Store 搜尋「SKS Spaces」→ 下載
  • Android：Google Play 搜尋「SKS Spaces」→ 安裝

👤 首次使用：帳號註冊

  1. 開啟 APP，點擊登入頁下方「註冊」
  2. 填入用戶名稱、Email（或手機號碼）
  3. 設定密碼（8–12 位，含大小寫英文、數字及特殊符號）
  4. 再次輸入確認密碼 → 點擊「確認」

🔑 一般登入 / 忘記密碼

  • 輸入帳號密碼 → 登入；或使用 Face ID / 指紋
  • 忘記密碼：點「忘記密碼？」→ Email 驗證 → 重設密碼
💡
契約用戶：帳號由工程人員建立，首次登入後請自行修改密碼。
左：登入頁　右：帳號註冊

3
保全操作

🤚 手動操作

啟動分區保全

  1. 點擊首頁對應分區的灰色盾牌圖示
  2. 系統自動檢查條件（門磁關閉、感測器正常）
  3. 條件符合 → 盾牌轉綠色，保全啟動
  4. 條件不符 → 顯示設備清單，確認後重試

解除保全：點擊綠色盾牌 → 轉灰色

解除警報：點擊紅色警示 → 恢復灰色

全區操作：底部導覽 → 服務管理 → 全區設定 / 全區解除

⚠️
啟動前請確認所有門窗已關閉，否則系統提示無法啟動並列出未關閉設備。

🕐 自動排程

路徑：服務管理 → 生效中的排程 → 新增排程

類型適用情境
每日固定每天相同時間啟動 / 解除，適合固定作息場域
每日獨立各天不同時間，彈性對應各班表
左：每日固定　右：每日獨立排程
4
攝影機監控與影像回放

📺 即時影像

  1. 底部導覽 → 影像管理
  2. 點擊攝影機名稱 → 開啟單路即時影像
  3. 手機橫向旋轉 → 最多同時 4 路顯示
  4. 長按卡片可拖曳調整排列順序

⏪ 影像回放

  1. 進入單一攝影機頁面 → 切換回放模式
  2. 時間軸拖曳選取時段
  3. 依事件篩選：人形偵測、移動偵測、門磁事件等

💾 錄影規格

類型天數說明
全時錄影（SD 卡）約 14 天256GB Micro SD
事件錄影（雲端）30 天前 10 秒＋後 20 秒，max 300 筆
事件錄影（SD 卡）滿後覆蓋前 10 秒＋後 20 秒
💡
雙重備援：雲端＋SD 卡同步，網路中斷仍持續錄影。
左：影像管理列表　中：橫向多路監視　右：影像回放與事件篩選
5
日誌查詢

路徑：底部導覽列 → 日誌管理

🔍 篩選選項

  • 事件類型（警報 / 操作 / 狀態）
  • 裝置類型（感測器 / 攝影機 / 主機）
  • 日期範圍、操作者帳號

📋 讀取技巧

  • 勾選「僅顯示保全事件」→ 快速聚焦警報紀錄
  • 📷 圖示 → 點擊直接查看對應影像片段
  • ⚠️ 保全事件高亮標示，一眼辨識

📌 紀錄內容

  • 保全啟用 / 解除（含操作者與時間）
  • 感測器觸發事件（設備名稱、時間、狀態）
  • 攝影機人形 / 移動偵測影像
  • 主機操作：備份、更新、網路切換
  • 非法刷卡紀錄
💡
日誌無資料時，先取消所有篩選條件並擴大日期範圍再查詢。
6
設備綁定

🖥️ 新增主機

  1. 首頁右上角 → ＋（新增設備）
  2. 選「主機」→ 輸入序號或掃描背面 QR Code
  3. 設定名稱與地點，完成網路設定

📡 新增感測器

  1. 主機管理 → 新增感測器
  2. 安裝電池，掃描感測器 QR Code
  3. 設定名稱、分區、連動攝影機
  4. 等候約 1.5 分鐘，LED 停止閃爍即成功

📷 新增攝影機（IPcam）

  1. 主機管理 → 新增攝影機
  2. 選連線方式（Wi-Fi 或 PoE）
  3. 依 APP 引導完成綁定

🔧 新增 WDI（有線無線轉換器）

  1. 接通 24V 電源
  2. 掃描 WDI QR Code
  3. 設定各乾接點對應的感測器名稱與分區
💡
QR Code 模糊時，開後蓋掃內蓋備用 QR Code。
首頁右上角 → 拓樸圖示，可視化查看所有設備架構。
7
授權管理

路徑：設定 → 授權

➕ 新增授權

  1. 點擊「新增」
  2. 掃描被授權人 QR Code，或輸入 Email
  3. 設定授權範圍（地點 / 分區）
  4. 選「永久授權」或指定到期日期

🏷️ 授權狀態

狀態說明
永久授權長期有效，無期限
授權至 yyyy-mm-dd到期自動失效
授權到期需重新授權
已停用手動停用，可重新啟用
8
主機網路設定

路徑：設定 → 設備列表 → 主機 → 網路設定

連線方式選擇畫面
Wi-Fi 選擇與密碼輸入

📶 Wi-Fi 無線網路

  1. 選「無線網路」
  2. 選擇 SSID → 輸入密碼 → 點「連接」
⚠️
主機僅支援 2.4GHz，5GHz 路由器請切至 2.4GHz。

🔌 有線 Ethernet

  • 選「有線網路」，網路線接 WAN Port
  • 可設定 DHCP 或 Static IP

📱 4G LTE 行動網路

  1. 插入 Nano SIM 卡，選「行動網路」
  2. 設定 APN、使用者名稱、密碼、Pin
  3. 點「下一步」完成
💡
LTE 僅在有線與 Wi-Fi 均斷線時自動啟用（備援機制）。

10
常見問題排除

📱 APP 操作問題

無法啟動保全
原因：門磁未關閉或設備離線
確認所有門窗已關閉，逐一確認清單中設備狀態後重試
攝影機畫面無法觀看
原因：攝影機離線或網路異常
確認電源與網路，點右上角重新整理，或重啟攝影機
APP 未收到推播通知
原因：手機通知被關閉
手機設定 → 通知 → SKS Spaces → 開啟允許通知
日誌頁面無資料
原因：篩選條件過嚴或日期範圍不符
取消所有篩選，擴大日期範圍後重新查詢

🔧 硬體問題

感測器無法入網（LED 持續閃）
原因：電池不足或 QR Code 模糊
更換 CR123A 電池；QR Code 模糊則開後蓋掃內蓋備用碼
緊急按鈕按下無反應
原因：電池接觸不良或設備死機
重新安裝電池；仍無反應則長按 Reset 10 秒恢復出廠設定後重新配對
讀卡機顯示「連線異常」
原因：主機網路中斷或訊號干擾
確認主機網路正常，縮短與主機距離；持續異常請聯絡客服
主機無法連接 Wi-Fi
原因：密碼錯誤或路由器使用 5GHz
重新執行 Wi-Fi 設定；5GHz 路由器請改用 2.4GHz
📞
聯絡新光保全客服時請提供：主機序號、發生時間、日誌截圖，以便快速協助。
